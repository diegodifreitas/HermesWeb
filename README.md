# Hermes Web 
### Work in process.
Sistema de software para gestão de parcerias entre a administração pública e as organizações da sociedade civil


## Getting Started

* npm install
* npm install -g json-server
* json-server --watch db.json --port 3001
* npm run start
* localhost:3000

### Prerequisites

* node 6+
* npm

## Running the tests

npm run test

## Built With

* React
* Redux
* Redux-form
* Json-Server
